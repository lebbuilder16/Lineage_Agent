# Lineage Agent

**Agentic Multi-Signal Forensics for Solana Token Families**

Lineage Agent is an open-source intelligence platform that reconstructs the full operational lineage of Solana tokens — from on-chain mint history to deployer identity, cartel detection, bundle analysis, and AI-powered risk scoring.

---

## What is Lineage Agent?

The Solana blockchain hosts over one million new token launches per month. A systematic exploitation pattern has emerged: operators deploy recognisable tokens, extract capital via rug pull or coordinated dump, then redeploy near-identical tokens targeting the same audience.

Lineage Agent exposes this pattern by correlating:
- **On-chain forensics** — wallet clustering, fund flow, bundle detection
- **Metadata DNA** — logo hashing, name/ticker similarity scoring
- **Operator profiling** — deployer track record, family tree reconstruction
- **AI analysis** — Claude-powered risk narrative with structured signals
